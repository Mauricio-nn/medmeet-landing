## :wrench:Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Medtech-Innovations/medmeet-landing
   ```
2. Install NPM packages in the cloned file
   ```sh
   npm install
   ```
3. Run the following command for the Tailwind changes
   ```sh
   npm run watch
   ```
